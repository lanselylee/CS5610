Rules

How to play

  1. Fill the grid so every row contains each number exactly once.
  2. Every column also contains each number exactly once.
  3. Every sub-grid (box) contains each number exactly once.

This site is a static mock for learning HTML & CSS only.

Tips for beginners

  • Start with the easy 6×6 puzzles
  • Look for rows, columns, or boxes with many numbers already filled
  • Use the process of elimination
  • Don't guess – use logic!
